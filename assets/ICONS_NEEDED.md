# Icon Assets Required

This directory needs the following icon files:

## extension-icon.png
- **Size**: 512x512 pixels
- **Format**: PNG with transparency
- **Description**: Main extension icon shown in Raycast preferences
- **Design suggestion**: A monitor/display icon, possibly with an arrow indicating movement between displays

## command-icon.png (optional)
- **Size**: 512x512 pixels
- **Format**: PNG with transparency
- **Description**: Command-specific icon (can use same as extension icon)
- **Design suggestion**: Same as extension icon or a simplified version

## How to Create Icons

You can:
1. Use SF Symbols app on macOS to export icons
2. Create custom icons using design tools (Figma, Sketch, etc.)
3. Use icon generator websites
4. For testing: Use any 512x512 PNG image

## Testing Without Icons

For development, Raycast will use a default icon if these files are missing.
The extension will still function correctly without custom icons.
